# Trippi repo notes

- `hermes/trippi.config.example.yaml` as a starter Hermes profile config
- `skills/hyatt-strategy/SKILL.md` for a Hyatt-first workflow
- Hyatt MCP wiring in `.mcp.json` and `opencode.json`
